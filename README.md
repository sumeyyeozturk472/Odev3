# Ödev 3
## Komik içerikler içeren web sitesi yapımı
* Anasayfa (index.html): genel site tanıtımı, sitede ne var, bölümler: komik resimler, videolar vs.
* Bölüm sayfaları (videolar.html, resimler.html, vs): komik yazılar, resimler ve videolar
* Kayıt sayfası (kayit.html): siteye kayıt sayfası (sadece görüntü), form ve form elementleri kullanılacak, form validasyon içerecek
* Giriş sayfası (giris.htm): siteye giriş sayfası (sadece görüntü), form ve form elementleri kullanılacak, form validasyon içerecek
* Site sayfalarının genel yapısı için semantic html elementleri kullanılacak
* Her sayfada aynı site adı/logosu ve navigasyon linkleri olmalı
* Öğeleri birbirinden ayırmak için div etiketi kullanabilirsiniz
* Web sitesi yayınlanıp linki readme dosyasına eklenecek
* Bütün ödev dosyaları ödev repo'sunda yer alacak
